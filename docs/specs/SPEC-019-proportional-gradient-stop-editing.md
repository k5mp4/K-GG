# SPEC-019: Gradient Rampのプロポーショナルストップ編集

## 背景・問題

Gradient Rampではストップをドラッグして移動できるが、周囲のストップとの間隔を保ちながら局所的に伸縮させる操作はできない。

## ゴール・成功条件

- Shiftを押してカラーストップをドラッグすると、選択ストップに近い未選択ストップほど大きく、遠いストップほど小さく移動する。
- ランプ範囲の外へストップを移動・削除せず、端を限度として位置を保つ。

## スコープ

### 対象

- サイドバーと拡大モーダルのカラーストップにおけるShift+ドラッグ。
- 複数選択されたカラーストップを一体として扱うプロポーショナル移動。
- 利用者向けの操作説明。

### 対象外

- 不透明度ストップ、ホイール操作、キーフレーム、プリセット形式の変更。
- プロポーショナル編集の半径や減衰曲線を設定するUI。

## 方針

- 選択ストップはドラッグ量の全量を移動し、未選択ストップは選択集合からの最短距離をランプ全長で正規化した二次減衰で移動する。これにより、近いストップほど強く追従する。
- ドラッグ対象ストップが0〜1（Mirror時は0〜0.5）の端へ到達した後は、その実移動量を使う。全ストップを同じ範囲へclampし、ストップの削除は行わない。
- Shift+クリックによる複数選択の既存操作は維持する。選択済みストップでは、クリック完了時だけ選択解除し、ドラッグした場合は選択を維持してプロポーショナル編集を行う。

## 受け入れ条件

- AC-001: 1つのカラーストップをShift+ドラッグすると、選択ストップがドラッグ量だけ移動し、未選択ストップは距離に応じてより小さく同方向へ移動する。
- AC-002: 複数選択時は選択した全ストップが同じ量だけ移動し、未選択ストップは最も近い選択ストップからの距離に応じて移動する。
- AC-003: ドラッグ対象がランプ端へ達しても、全カラーストップは編集可能範囲内に残り、削除されない。
- AC-004: サイドバーと拡大モーダルで同じ操作結果になる。通常ドラッグ、不透明度ストップ、Shift+クリックの複数選択は既存どおり利用できる。
